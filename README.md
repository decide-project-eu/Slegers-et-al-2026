# Slegers-et-al-2026
Code scripts for analyses belonging to the manuscript of Slegers et al. (2026)

Analyses are done in R. The dataset that is loaded is based on the work in Slegers et al. (2024) ([link to GitHub page](https://github.com/decide-project-eu/slegers-et-al-2024)). The following R packages are used: performance_0.16.0, ggeffects_2.3.2, ggplot2_3.5.2, glmmTMB_1.1.14, lme4_1.1-37, Matrix_1.7-1, dplyr_1.1.4
